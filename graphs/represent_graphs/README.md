- ## **Способы представления графов**

    |                      | Список ребер    | Матрица смежности | Список смежности |
    | -------------------- | --------------- | ----------------- | ---------------- |
    | **Используемая память**  | $\theta(n^{2})$ | $O(n^{2})$        | $O(n^{2})$       |
    | **Вставка, удаление, изменение ребра** | $O(1)$          | $O(1)$            | $O(1)$           |
    | **Перебор соседей вершины a** | $O(1)$          | $O(1)$            | $O(1)$           |
