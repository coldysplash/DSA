- ## **Способы представления графов**

    |                      | Список ребер    | Матрица смежности | Список смежности |
    | -------------------- | --------------- | ----------------- | ---------------- |
    | **Используемая память**  | $O(E)$ | $O(V^{2})$        | $O(V+E)$       |
    | **Вставка, удаление, изменение ребра** | $O(E)$          | $O(1)$            | $O(deg a)$           |
    | **Перебор соседей вершины (a)** | $O(E)$          | $O(V)$            | $O(deg a)$           |
